##### Open Terminal
```
Ctrl + Alt + T

# On Debian
Super(Windows) + T
```

##### Tile Windows
```
Super + Arrow
```

##### Terminal Font Size
```
# Larger
Ctrl Shift +

# Smaller
Ctrl -
```

##### Clear Terminal
```
Ctrl + L
```

##### Break Process
```
Ctrl + C
```

##### Background/Suspend Process
```
Ctrl + Z
```

##### Autocomplete
```
Tab
```

##### Close Terminal
```
Ctrl + Shift + W
```
