# Enumerate All
```bash
enum4linux -a <IP Address>
```

# Enumerate OS
```bash
enum4linux -o <IP Address>
```

# Enumerate Shares
```bash
enum4linux -S <IP Address>
```

# Enumerate Users
```bash
enum4linux -U <IP Address>
```
